# audit-grade-ai-workstation

This repository contains research artifacts and design rationale for audit-grade AI safety evaluation.

## Veritas Ω

The Veritas Ω admissibility framework and related audit materials are located in:

- `veritas-omega/`

These documents define structural criteria for determining whether high-stakes AI systems are admissible for deployment.

This repository is released as-is. No roadmap or support is implied.

## Engagement Model

EasyStreet engagements are authorization-based, not advisory.

If you can provide a folder, repository, or system boundary you are authorized to test, EasyStreet will return a sealed, verifiable evidence package produced under the AEGIS-ALD-W1 protocol.

No demos. No opinions. No iterative consulting.
