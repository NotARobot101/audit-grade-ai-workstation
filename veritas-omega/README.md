# Veritas Ω Framework

This directory contains the canonical specification, methodology, audits,
and executive materials for the Veritas Ω admissibility framework.

## Structure

- `spec/` — canonical admissibility specification
- `methodology/` — signature extraction and derivation discipline
- `audits/` — hostile evaluations of real-world systems
- `briefs/` — executive and deployment-facing summaries

All materials are released as-is for research and evaluation.
